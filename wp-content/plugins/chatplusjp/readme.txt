Add ChatPlus to the page.

== Description ==
This plugin can automatically add "Chat Plus" to your page. This allows you to insert "Chat Plus" briefly without editing the theme.

== Installation ==

You can install this plugin directly from your WordPress dashboard:
 
 1. Go to the *Plugins* menu and click *Add New*.
 2. Search for *chatplusjp*.
 3. Click *Install Now* next to the *chatplusjp* plugin.
 4. Activate the plugin.
